## Portfolio-maylis

Projet portfolio qui présente les travaux réalisés au cours de mes différentes formations en développement web.

## Installation
1. Clonez le dépôt
git clone https://github.com/Gaillard05/portfolio-maylis.git
cd portfolio-maylis
2. Installez les dépendances
composer install
3. Configurez votre base de données dans le fichier .env. Par exemple:
DATABASE_URL="mysql://db_user:db_password@127.0.0.1:3306/db_name?serverVersion=5.7"
Démarrez le projet sur un serveur local
